# CELTIC - Collaborative Environment for Learning and Teaching with Integrated Coding

**The project's goal is to create a collaborative programming tool that allows students to do programming on a shared whiteboard. Students will be able to connect to the board using their own small computer module (a Docker container) and execute programming exercises that can be visualized on the whiteboard alongside their peers.**

## Getting Started

### Prerequisites

- Node.js and npm
- TypeScript
- Express
- Nodemon
- ts.Node
- mongodb
- mongoose

### Installation

```bash
# Clone the repository
git clone https://github.com/your-username/your-repo.git

# Navigate to the project directory
cd your-repo

# Install client-side dependencies
cd client
npm install

# Install server-side dependencies
cd ../server
npm install
